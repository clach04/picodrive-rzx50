SDL PicoDrive

Small semi-portable Sega MegaDrive/Senesis ported to SDL by dmitrysmagin.
https://github.com/dmitrysmagin/picodrive-rzx50

To build, find a Makefile under the platform tree. For example:

    platform/sdl

The SDL platform is the main focus of this repository.
It is unknown whether platform/linux or platform/dingux work, working
Linux desktop and OpenDingux is in the SDL platform.
